# Login Form Project

A professional and responsive login page built with HTML and CSS.

## Features

- **User ID Input Field** - Secure field for entering username or email
- **Password Input Field** - Hidden password input for security
- **Login Button** - Submit button to authenticate users
- **Sign Up Link** - Quick navigation for new users to create an account
- **Responsive Design** - Mobile-friendly layout that works on all screen sizes
- **Modern Styling** - Clean UI with gradient background and smooth transitions
- **Form Validation** - Required field validation on client-side

## Technologies Used

- HTML5
- CSS3
- JavaScript (for form handling)

## Files Included

- `index.html` - Main login page with form and styling
- `signup.html` - Sign up page (linked from login form)

## How to Use

1. Open `index.html` in your web browser
2. Enter your User ID in the first field
3. Enter your Password in the second field
4. Click the **Login** button to submit
5. New users can click the **Sign Up here** link to create an account
